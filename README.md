# iPhone 17 Model (HTML & CSS)

A fully built iPhone 17 model created entirely with HTML and CSS. Every detail is handcrafted with code. No images, no JavaScript, just pure HTML & CSS magic.

## How to Explore
- Open this file in any web browser.
- Inspect each element to see the structure and CSS styling.
- Ideal for learning advanced CSS positioning, shadows, gradients, and handcrafted shapes.
